# muse-status

Shamelessly the most beautiful status command for your Linux
setup.

`muse-status` can provide output for the following formats:

-	`lemonbar`

-	`i3bar`/`swaybar`

-	Pango markup

# Building

The following (Void Linux) dependencies are required to build `muse-status` and
its daemon:

-	pkg-config

-	alsa-lib-devel

-	dbus-devel

-	openssl-devel
